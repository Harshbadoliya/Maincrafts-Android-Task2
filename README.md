# 📱 Maincrafts Android Internship – Task 2

## 📝 Notes App (Room Database + MVVM + Jetpack Compose)

This project is developed as part of the **Maincrafts Android Internship – Task 2**.

The application is a Notes App built using modern Android development practices with:

- 🏗 MVVM Architecture
- 🗄 Room Database (Local Persistence)
- 🔄 Kotlin Flow
- 🎨 Jetpack Compose UI
- 🧠 Repository Pattern

---

## 🚀 Features

- ➕ Add new notes
- 🗑 Delete notes
- 💾 Persistent data using Room Database
- 🔄 Data survives app restart
- 📱 Clean Compose UI
- 🏗 Proper MVVM architecture

---

## 🏛 Architecture Flow

Database → Repository → ViewModel → UI

This ensures:
- Clean separation of concerns
- Lifecycle awareness
- Scalable architecture
- Testable code

---

## 🛠 Tech Stack

- Kotlin
- Jetpack Compose
- Room Database
- ViewModel
- StateFlow
- Material 3
